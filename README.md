# Gamex OS

Gamex OS is a simple command line shell meant as a starting point for a
games-focused operating environment. It includes a few built-in
commands and a tiny demo game so you can experiment with the system.

## Running

```bash
python3 src/main.py
```

Once running, type `help` to see available commands. You can list games
with `games` and start one with `play <game>`. Currently the only built-
in game is `guess`, a number guessing challenge.
